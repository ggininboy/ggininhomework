## 基本操作
```
$ ls
```
可以利用ls來確認目前資料夾中有那些檔案。
```
$ cd [欲進入之資料夾]
```
確認好欲進入之檔案，可利用cd進入資料夾。
```
$ vim [欲進入之檔案]
```
利用vim可進到.sh檔或者.py檔裡面看程式碼，欲修改程式碼可以按"i"來對程式碼進行修改，修改完要記得輸入":wq"來進行儲存並退出。
```
$ sh [欲執行之.sh檔]
```
當我們想要執行.sh檔時，我們可以利用sh後面加欲執行之.sh檔來執行。
